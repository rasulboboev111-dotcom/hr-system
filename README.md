# Smart Staffing — Интеллектуальная HR-платформа

Интегрированная система управления персоналом (HRM), разработанная на базе **Laravel 9**, **Vue.js 3** и **Inertia.js**. Платформа автоматизирует ключевые HR-процессы: от учета посещаемости и расчета зарплат до интеллектуального анализа кадров с помощью AI.

## Основные возможности
*   **Управление сотрудниками**: Полная база данных с возможностью импорта из CSV.
*   **Учет посещаемости (Timesheet)**: Автоматизированный табель с расчетом рабочих часов.
*   **Расчет зарплаты (Payroll)**: История выплат, бонусы и экспорт ведомостей.
*   **AI Advisor**: Интеллектуальные рекомендации по подбору и обучению персонала.
*   **Архив**: Хранение данных об уволенных сотрудниках.
*   **RBAC**: Гибкие роли и права доступа для администраторов и сотрудников.
*   **Мультиязычность**: Поддержка таджикского, русского и английского языков.

## Технологический стек
*   **Backend**: PHP 8.1+, Laravel 9.
*   **Frontend**: Vue.js 3 (Composition API), Vite, Inertia.js.
*   **Database**: PostgreSQL.
*   **Styling**: Tailwind CSS.

## Как запустить проект (Установка)

### 1. Требования
*   PHP 8.1 или выше.
*   Node.js (LTS).
*   Composer.
*   PostgreSQL.

### 2. Шаги установки
1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL_РЕПОЗИТОРИЯ>
    cd <ПАПКА_ПРОЕКТА>
    ```

2.  **Установите зависимости PHP:**
    ```bash
    composer install
    ```

3.  **Установите зависимости Node.js:**
    ```bash
    npm install
    ```

4.  **Настройка окружения:**
    *   Создайте файл `.env` (если его нет) на основе `.env.example`:
        ```bash
        cp .env.example .env
        ```
    *   Настройте подключение к базе данных PostgreSQL в блоке `DB_...`

5.  **Генерация ключа приложения:**
    ```bash
    php artisan key:generate
    ```

6.  **Запуск миграций (создание таблиц):**
    ```bash
    php artisan migrate
    ```

7.  **Сборка фронтенда:**
    ```bash
    npm run build
    ```

8.  **Запуск сервера:**
    ```bash
    php artisan serve
    ```

Теперь проект доступен по адресу: `http://localhost:8000`
